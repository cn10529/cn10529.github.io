如何下载和安装 Java7                              

一、前言

关于下载 Java7 ，在 Java 官网找到的是这个：

既然是这样的话那我们只能到其他网页下载，比如 Oracle 官网（其实 Java 只是它其中的一个产品，在这里也能搜索到关于 Java 历史版本，缺点是登录）

二、下载

这是 Oracle 官网关于 Java7 下载界面：

https://www.oracle.com/cn/java/technologies/javase/javase7-archive-downloads.html

这是 Oracle 官网关于 Java7 下载的地址：

https://download.oracle.com/otn/java/jdk/7u80-b15/jdk-7u80-windows-x64.exe

但是需要登录，你可以登录，但我最烦登录了，所以我到其他网页下载。

这个是华为镜像站的 Java 界面：

https://repo.huaweicloud.com/java/

你也可以在必应直接搜索“华为镜像站”然后再在网址后面添加“ java/ ”。

界面：



点击“ jdk/ ”



然后点击“ 7u80-b15/ ”



这里是要看你的系统位数，如果你的系统是 32 位那你就选择“ jdk-7u80-windows-i586.exe ”，如果你的系统是 64 位那你就选择“ jdk-7u80-windows-x64.exe ”，然后就可以下载了。


三、安装



打开文件



点击“ 下一步 (N) > ”



这一步是要你是否要安装到自定义位置的，如果要那你就点击“ 更改 (C)… ”，如果不要就直接点击 “ 下一步 (N) > ”就开始安装了 。



然后点击“ 关闭 (C) ”完成安装。
